# QBERT Quantization on Transformer Models

## Overview

This repository contains my internship work on the implementation and evaluation of QBERT quantization on multiple Transformer-based language models:

* TinyBERT
* MobileBERT
* ALBERT
* BERT-Base
* DistilBERT

* 16 GB GDDR6 GPU Memory
* CUDA-enabled environment

Additional experiments were executed on the following GPU:

* NVIDIA L40S
* 46 GB GPU Memory
* CUDA Version: 13.0
